## Boba Drinks & Ice-cream App

An app you can use to order for boba tea and ice-cream.

## Current Features
* Select through various flavors of boba teas and ice-cream

* Add to cart

* Time order was made

* Checkout

## Installation

1. Clone repo
2. run `npm install`

## Usage

1. run `node server.js`
2. Navigate to `localhost:4040`
